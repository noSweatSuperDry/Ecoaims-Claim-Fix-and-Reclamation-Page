# EcoaimsClaimFixAndReclamations
This is a CRUD project to complete Degree of Riveria

How to install:

1. Go to server folder and run scripts.
Go to server folder
### cd ./server
Install dependencies 
### npm install 
Start server
### npm run devStart
 
Backend initiated!

2. Goto client folder and run scripts

Go to client folder
### cd ./client
Install dependencies 
### npm install 
Start server
### npm start
